Acerca de CineMundo

Tecnología y Pasión por el Séptimo Arte.

CineMundo nació como un proyecto académico y personal con el objetivo de demostrar que el diseño web moderno y el amor por el cine pueden convivir en una sola plataforma. Desarrollado por un estudiante de informática apasionado por la web dinámica, este sitio utiliza tecnologías como:

  • HTML5 Semántico y CSS Moderno (Flexbox & Grid)
  • Lógica interactiva con JavaScript y jQuery
  • Diseño responsivo para dispositivos móviles
  • Experiencia de usuario optimizada con SweetAlert2 y Select2

Desarrollado con dedicación en Guatire para el mundo.
